# ファイルアップロードアプリ  
フロントエンドだけでファイルアップロードアプリのドラッグアンドドロップでのアプリの実装をしてみた  

## 変更したところ  
style.cssで黒基調のモダンっぽいUIにした  

## 学んだこと  
npm init -y  
npm install -D typescript vite  
npx tsc --init  
これらのコマンドは覚えておくと素早くフロントの環境を構築できる  

DOM操作でファイルのアップロードのフロントだけの流れ  
pointe-eventsでドラッグ&ドロップのクラスの割り当てが制御できる    

あえてinputタグを消してlabelタグにファイルアップロードボタンを担ってもらうことで見た目が整えやすい
