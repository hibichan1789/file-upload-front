# ファイルアップロードアプリ  
フロントエンドだけでファイルアップロードアプリのドラッグアンドドロップでのアプリの実装をしてみた  

## 変更したところ  


## 学んだこと  
npm init -y  
npm install -D typescript vite  
npx tsc --init  
これらのコマンドは覚えておくと素早くフロントの環境を構築できる  


## 所感  
typescriptのパッケージのコマンド、gitのコマンドが割とスムーズにできている  
できるだけマウスに頼らずに、コードを書けている気がする、これは先週からコマンドやショートカットを使うことを意識しているからかも
